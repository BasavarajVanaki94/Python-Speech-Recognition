## Dependancies

Now you would need to Install all the Dependecies to begin running
playing the Examples 

Linux users  
----------

```bash
 
$ pip3 install pydub
$ pip3 install PyAudio
$ pip3 install SpeechRecognition
```

Window users
-----------

```bash 
$ pip install pydub
$ pip install PyAudio
$ pip install SpeechRecognition
```
